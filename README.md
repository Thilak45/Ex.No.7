# Aim: To develop a prompt-based application using ChatGPT - To demonstrate how to create a prompt-based application to organize daily tasks, showing the progression from simple to more advanced prompt designs and their corresponding outputs.

#AI Tools Required: 
ChatGPT

# Explanation: 
Prompt:
"Design a personal productivity assistant that can help manage daily tasks, schedule reminders, suggest wellness tips, and answer general queries. The assistant should interact using natural language and be adaptable to the user’s changing preferences over time."
Procedure:
1. Define the core requirements of a personal productivity assistant.
2. Identify and construct appropriate prompts for each task using an LLM (e.g., ChatGPT).
3. Simulate natural user interaction through a simple interface or command-line system.
4. Collect feedback or inputs from users and adapt responses accordingly.
5. (Optional) Integrate basic memory to simulate preference adaptation.
EXPECTED OUTPUT: - (attached the drive link)
Output (Example Response by LLM):
Personal Productivity Assistant Features:
1. Daily Task Manager:
o Accept tasks via natural language (e.g., "Remind me to call mom at 6 PM").
o Organize tasks by priority and deadline.
o Provide daily summaries and pending items.
2. Smart Scheduler:
o Schedule events and set reminders using contextual understanding.
o Notify user of overlapping appointments or free time slots.
3. Wellness Tips Generator:
o Suggest daily wellness advice (hydration, exercise, screen-time breaks).
o Adapt suggestions based on past user preferences and responses.

**Procedure** 
**1. Define the Core Requirements**

The following requirements were identified for the productivity assistant:

**Functional Requirements**
Add and manage daily tasks
Schedule reminders and appointments
Provide wellness and productivity tips
Answer general-purpose questions
Maintain conversational interaction
Adapt to user preferences

**Non-Functional Requirements**
User-friendly interaction
Fast response generation
Natural language understanding
Cross-platform accessibility

**2. Construct Appropriate Prompts**

Different prompts were designed for various assistant functionalities.

**Task Management Prompt**

You are a smart productivity assistant. Help users create, organize, and prioritize daily tasks based on deadlines and importance.

**Scheduling Prompt**

You are a scheduling assistant. Understand natural language requests and create reminders, meetings, and event schedules while avoiding conflicts.

**Wellness Prompt**

You are a wellness coach. Suggest healthy habits, hydration reminders, exercise tips, and stress-management advice personalized to the user’s lifestyle.

**System Design**
Modules of the Assistant
Module	Function
Task Manager	Stores and organizes tasks
Smart Scheduler	Creates reminders and schedules
Wellness Advisor	Generates health and productivity tips
Query Handler	Answers general user questions
Preference Memory	Adapts responses based on user behavior
Simulated User Interaction
**Example 1 – Task Management**
User Input

“Remind me to submit my assignment tomorrow at 5 PM.”

Assistant Response

“Reminder created: Submit assignment tomorrow at 5:00 PM.”

**Example 2 – Scheduling**
User Input

“Schedule a meeting with my project team on Friday at 3 PM.”

Assistant Response

“Meeting scheduled for Friday at 3:00 PM with your project team.”

**Example 3 – Wellness Suggestion**
User Input

“I’ve been working continuously for 4 hours.”

Assistant Response

“You should take a short 10-minute break, drink water, and stretch your body to reduce fatigue.”

**Example 4 – General Query**
User Input

“What is the weather today?”

Assistant Response

“The weather today is partly cloudy with a temperature of 28°C.”
